Welcome to Algnite Voice Navigator!

Thank you for installing our voice navigation extension. This tool will help you navigate websites using voice commands, making web browsing more accessible and convenient.

Key Features

Voice Commands

Control your browser using natural voice commands. Here are some examples:

  • "Click [element name]" - Click on any element on the page
  • "Scroll up/down" - Navigate through the page
  • "Go to [link text]" - Click on a specific link
  • "Read [element name]" - Have the text read aloud
  • "Stop" - Stop voice control

Element Highlighting

Enable element highlighting to see which elements are being targeted by your voice commands. This helps you understand what the extension is focusing on.

Customization

Adjust the voice speed and highlight color to match your preferences. These settings are saved across sessions.

Getting Started

To start using the extension:

  1. Click the extension icon in your browser toolbar
  2. Click "Start Voice Control" or use the keyboard shortcut Ctrl+Shift+Space
  3. Start speaking commands naturally
